League Invaders

by: Dave Dunn

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I like space and rockets. I also dislike aliens. I figured a game where I fly a rocket and shoot aliens would help me relieve some stress. I also wanted to pay tribute to the game Galaga because it is one of my favorites.

2. What I Learned

I learned that people like a variety of enemies in their games. The same enemies repeating the same pattern quickly gets old. I should vary the gameplay to prevent it from getting stale and boring.

3. Architecture of My Game

The heart of the game is the GamePanel class. That controls when all the objects should draw and update. The objects are the Rocket, the Aliens, and the bullets. Every time they collide, the ObjectManager class is responsible for either removing the object from the game or signaling to the GamePanel class that the game is over. At that point, the GamePanel class will draw the game over screen.

4. Technical Implementation

The collision in the game is accomplished by iterating through all of the objects on the screen for each frame of animation. The ObjectManager class checks which objects are overlapping by comparing the horizontal position, vertical position, width, and height of each object. Based on the types of overlapping objects, the ObjectManager class will either remove the objects, signal a game over state, increase the score, or do nothing at all. This happens approximately sixty times per second.

5. Most Challenging Part

The most challenging part of this project was remembering what each part of the program was responsible for. This was the first time I worked on a project of this size. When problems would occur, it was not immediately obvious where I should start troubleshooting.

6. Future Fixes and Updates

I would add more enemies and different enemy patterns. I would also like to implement a power-up system. The player would be able to acquire special blasters, bombs, and shields to handle the different enemy types.
